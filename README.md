#  🎮 mcMMO for Minecraft

<pre>
🚀 핵심 기능 (Key Features)</br>
1. 4대 전문 스킬 (Core Skills)</br>
모든 스킬은 전용 도구를 착용했을 때만 활성화되며, 레벨에 따라 능력이 선형적으로 강화된다.</br>

    스킬	전용 도구	  주요 능력 (Abilities)	                마스터리 (Lv.30)</br>
------------------------------------------------------------------------------</br>
⛏️ 채광	    곡괭이	      레벨당 1% 확률로 2배 드랍	            3x3 범위 채광</br>
🪓 벌목	    도끼	      BFS 기반 지능형 벌목	                트리 펠러 (웅크리기 시)</br>
🧹 삽질	    삽	          레벨별 희귀 보물 발견 확률 증가	        보물 발견 확률 극대화 (45%)</br>
⚔️ 전투	    칼	          레벨당 데미지 +0.5 & 크리티컬 확률	    데미지 +15 & 크리 30%</br>

- 정교한 레벨링 밸런스</br>
1. XP 차등 지급: 광물의 희귀도, 몹의 등급에 따라 획득 경험치가 세밀하게 조정되어 무분별한 레벨업을 방지</br>
2. 어뷰징 방지: 가축 사냥 시 경험치를 대폭 하향하고, '막타(Kill)' 보너스를 강화하여 실질적인 사냥의 재미를 극대화</br>

- 인터페이스</br>
1. 고정형 스탯 GUI: /stats 명령어로 열리는 창은 아이템 유출 및 이동이 불가능한 안전한 인터페이스 제공</br>
2. 액션바(Action Bar)를 통한 경험치 획득 알림과 레벨업 시 화면 중앙 타이틀 연출 지원</br>
3. 관리자 알림: 관리자가 레벨을 조정할 경우 해당 플레이어에게 즉시 안내 메시지가 가도록 설정</br>

🛠️ 명령어 가이드 (Commands)</br>
👤 유저 명령어</br>
/stats : 자신의 스킬 레벨과 경험치, 고유 능력을 확인할 수 있다.</br>
또한, Shift + F 를 통한 스텟 부여가 가능하다.</br>

👑 관리자 명령어</br>
/mcmmo set <플레이어> <스킬> <레벨> : 대상의 특정 스킬 레벨을 강제로 설정다.</br>

스킬 종류: MINING, WOODCUTTING, EXCAVATION, COMBAT</br>

/mcmmo reset <플레이어> : 대상의 모든 데이터를 초기화하고 DB에 반영한다.</br>

📝 설치 및 설정</br>
1. Intellij 의 우측 gradle 탭에서 Tasks > shadow > shadowJar을 찾아 더블 클릭한다.</br>
2. 생성된 jar 파일을 프로젝트 파일 > build > libs 에서 찾아 마인크래프트 서버 plugis 폴더에 넣는다.</br>

⚠️ 주의사항</br>
- 본 플러그인은 MySQL 데이터베이스 연동이 필수입니다. jar 파일을 빌드하기 전, 반드시 McMMO.java 파일의 20번 라인을 확인하세요(localhost: 본인의 DB 주소로 수정 & your password 부분을 본인 DB 비밀번호로 수정하세요)</br>
